Logical Model: Document model

Official URL: https://www.ehealth.fgov.be/standards/fhir/childrecord/StructureDefinition/BeDocument Version: 0.1.0
Draft as of 2025-08-11 Computable Name: BeDocument

EHDS refined base model for common document data elements, including the common header. Data relevant to document type and its content for administrative and searching purposes.

Usages:

  • Derived from this Logical Model: Child Report Logical Model
